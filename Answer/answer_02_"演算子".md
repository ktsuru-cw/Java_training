
# answer_02

長方形の面積を求めて画面に表示するプログラムを作成してください。  
(例1)
```java
public class Main {
 
    public static void main(String[] args) {
        int width = 3;
        int height = 5;
        int menseki = width * height;
        
        System.out.println(menseki);
    }
    
}
```
