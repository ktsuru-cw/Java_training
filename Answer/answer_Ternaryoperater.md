## Ternaryoperator_③
●以下のif文を三項演算子に直してください。

```java
// if文

if (drink.equals("お茶")) {
  System.out.println("これはお茶です");

  System.out.println("これはコーヒーです");
} else if (drink.equals("ジュース")) {
```
